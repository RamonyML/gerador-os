Instruções de Uso

Siga estas orientações para utilizar corretamente os modelos de O.S:

  1. Selecione a categoria adequada para sua necessidade
  2. Escolha o modelo mais apropriado para a situação
  3. Personalize o texto substituindo os campos em maiúsculas (XXXXX) pelos dados reais
  4. Revise cuidadosamente antes de utilizar a O.S
  5. Para situações não previstas, adapte o modelo mais similar
